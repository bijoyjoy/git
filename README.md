# 📥 Install Git

| Operating System | Command |
|-----------------|---------|
| 🪟 Windows | Download from https://git-scm.com/downloads |
| 🍎 macOS | `brew install git` |
| 🐧 Ubuntu/Debian | `sudo apt update && sudo apt install git` |

---

## Verify Installation

```bash
git --version
```

Output

```bash
git version 2.50.1
```
> 💡 **Tip:** Always verify the installation using `git --version`.
